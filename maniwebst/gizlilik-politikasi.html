🔒 Gizlilik Politikası

Kişisel verilerinizin korunması bizim için önemli

📋 Genel Bilgiler

Manifest.com olarak, kullanıcılarımızın gizliliğini korumayı taahhüt ediyoruz. Bu gizlilik politikası, web sitemizi ziyaret ettiğinizde toplanan bilgilerin nasıl kullanıldığını açıklar.

Son güncelleme: 2024

İletişim: maniwebst@gmail.com

📊 Toplanan Bilgiler

🤖 Yapay Zeka İşlemleri

Manifest özelliğimizi kullandığınızda, yazdığınız metin OpenAI API'sine gönderilir. Bu veriler:

  • Sadece manifest yanıtı oluşturmak için kullanılır
  • Sunucularımızda saklanmaz
  • Üçüncü taraflarla paylaşılmaz
  • İşlem tamamlandıktan sonra silinir

🌐 Web Sitesi Kullanımı

Web sitemizi ziyaret ettiğinizde aşağıdaki bilgiler otomatik olarak toplanabilir:

  • IP adresiniz (genel konum bilgisi için)
  • Tarayıcı türü ve sürümü
  • İşletim sistemi
  • Sayfa ziyaret süreleri
  • Tıklanan bağlantılar

🍪 Çerezler (Cookies)

Web sitemizde çerezler kullanılmaktadır:

📱 Gerekli Çerezler

  • Site fonksiyonlarının düzgün çalışması için
  • Kullanıcı tercihlerini hatırlamak için
  • Güvenlik amaçlı

📊 Analitik Çerezler

  • Site kullanım istatistikleri için
  • Performans iyileştirmeleri için
  • Kullanıcı deneyimini geliştirmek için

🔐 Veri Güvenliği

Kişisel verilerinizi korumak için aşağıdaki önlemleri alıyoruz:

  • HTTPS şifreleme protokolü kullanımı
  • Güvenli sunucu altyapısı
  • Düzenli güvenlik güncellemeleri
  • Veri erişim kontrolleri

📤 Veri Paylaşımı

Kişisel verileriniz aşağıdaki durumlar dışında üçüncü taraflarla paylaşılmaz:

  • Yasal zorunluluk durumunda
  • Güvenlik tehditlerine karşı koruma için
  • Hizmet sağlayıcılarımızla (sadece gerekli bilgiler)

👤 Kullanıcı Hakları

KVKK kapsamında aşağıdaki haklara sahipsiniz:

  • Kişisel verilerinize erişim hakkı
  • Verilerinizin düzeltilmesi hakkı
  • Verilerinizin silinmesi hakkı
  • İşleme kısıtlama hakkı
  • Veri taşınabilirliği hakkı
